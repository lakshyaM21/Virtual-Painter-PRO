# Virtual Painter Pro

A real-time, hand-gesture-controlled virtual painting application built with
**Python 3.13**, **OpenCV**, and the latest **MediaPipe Tasks API**
(`HandLandmarker` with `hand_landmarker.task`). Paint on your webcam feed.
using nothing but your hand вЂ” no mouse, no stylus.

> This project uses only the modern MediaPipe Tasks API. The deprecated
> `mp.solutions.hands` module is **not** used anywhere in this codebase.

---

## Features

- Real-time webcam hand tracking (MediaPipe `HandLandmarker`, Tasks API)
- Smooth, jitter-free drawing via exponential-moving-average point smoothing
- Brush and eraser tools
- Header-based color palette / tool selection bar
- Draw with your index finger, select tools with index + middle finger
- Save your painting to disk (gesture or keyboard)
- Clear canvas (gesture or keyboard)
- Undo last stroke (keyboard)
- Live FPS counter
- On-screen current mode and current color indicators
- Clean, alpha-blended canvas compositing with anti-aliased strokes
- Fully configurable via a single `config.py`
- Auto-creates the `Output/` folder if missing
- Type-hinted, documented, PEP8-compliant, object-oriented codebase

---

## Hand Gestures

| Gesture                        | Action           |
|--------------------------------|------------------|
| Index finger up only           | Draw             |
| Index + middle finger up       | Selection mode (choose tool/color from header) |
| Thumb up only                  | Save painting    |
| Closed fist                    | Clear canvas     |

## Keyboard Shortcuts

| Key   | Action        |
|-------|---------------|
| `S`   | Save painting |
| `C`   | Clear canvas  |
| `Z`   | Undo          |
| `ESC` | Exit          |

## Installation:-

1. Clone or download this project.
2. Ensure `Header/1.jpg` вЂ“ `Header/4.jpg` and `models/hand_landmarker.task`
   are present (these are not generated by this project and must already
   exist in the folders above).
3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

---

## Running

```bash
python main.py
```

A window titled **"Virtual Painter Pro"** will open showing your webcam
feed with the header tool bar overlaid at the top. Raise your index finger
to draw, or raise index + middle fingers and move over the header to pick
a color or the eraser.
